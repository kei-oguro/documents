7/15 火
-------

8:00の目覚ましで9:30起床。

水やりは10:00にペットボトル1本。

`まいばすけっと`に昼飯の材料の買い出しに。店を出たら結構強い雨が降り出して驚いた。数分で弱くなった。

__コーヒー__ を円錐フィルターで淹れた。`筋金珈琲焙煎所`のスジガネブレンド。

リモート出社。

__ランチ__ は焼きそばと袋キャベツ。昨日入れ忘れた焼きそばの粉末調味料と、今朝まいばすで買った焼きそば生麺を炒めた。具は冷凍豚肉とピーマンとシメジ。豚肉は解凍して塩コショウして2時間冷蔵庫で寝かせたもの。

ワイヤレス充電器がかなり熱くなる。初期不良化と思って元々自宅にあった方を使ってみても、やはり熱くなる。そうだっけ？距離が変わったり、MagSafe 化金属板を付けたせいかも？

知らなかった。 ["GPU がテクスチャをサンプリングするとき。GPU は現在のピクセルのテクスチャ座標 (UV) と GPU が計算する 2 つの内部値 DDX と DDY に基づいて、どのミップレベルを使用するかを決定します"](https://docs.unity3d.com/ja/2021.3/Manual/texture-mipmaps-introduction.html) でも、Unityではテクスチャインポート設定毎なんだね。モデル毎にミップマップバイアスを変えられないんだね。 >>> そうか、 tex2Dlod の引数に渡すだけか。

GitHub の Notifications はいつまでも残り続けるワケじゃないのか。半年くらいのようだ。

__晩飯__ はキムチ炒めとパックご飯と卵。炒め物は冷凍挽肉とシメジとモヤシとピーマンを入れて、ラードで炒めて、八丁味噌を追加して少し煮て、胡椒とごま油を追加。

- 届いて放置していた122本ドライバーセットを空けたら、普通のプラスとマイナスは小さいのしかない。
  - 容器が壊れた100均のドライバーセットを置き換えられるかと思ったが、場所を取る精密ドライバーセットになっただけ。
    - Amazonでドライバーセットを見てみると、今の100均のはネジになってるのがあるのはポイント高い。不具合は容器だけなので、なんとかできるといいんだけど。ジップロックに入れるだけでいいかも？
  - 中学か高校の時に買った、ドライバー以外使わない、小さなトンカチや切れないカッターやノコギリの入った工具セットは、ようやく捨てることにする。

[SOXAI](https://soxai.co.jp/)というところのスマートリングの記事を読んで見に行ってみた。僕の使ってる Apple Watch だと、エクササイズモードにすると電池消費が速過ぎる。スマートリングで心拍数をずっと測ってくれるなら試したい。と思っていたところ、スマートリングは睡眠時に色々計測することで睡眠の質を知る、という用途で人気があるらしい。安いものだと2000円くらいからある中で、この製品は、脈拍から呼吸数を得られること、体温を計測すること、血中酸素濃度を計測することが出来るところが利点だ。心拍数だけなら安いのでもいいんだけど。それと、安い商品でもストレスを測定できると言ってるが、それはどういう理屈なのだろう？

__胡椒__ の苗が届いていた。

Amazon Mastercard が届いていた。セブンイレブンだと7%(普段の1.5%に追加5.5%)というのは凄い。スーパーを使えばそれより安いかも知れないが、個人的にはセブンの利用頻度が高い。他で使う時は、PayPay のステップアップのために PayPay カード優先かな。Mastercard の方が加盟店手数料が安いことがあるらしいから、30回利用済みで気付けはそっち。元々、使える時は PayPay だけど。

__FX__ はCPIが予想以上だったのでドル高。僕のAUD/USDの含み損が減っている。そろそろ損切りすべき？三度目の撤退チャンス。
