- <details><summary>SIMDの行列演算</summary>

  左から掛ける
  ```glsl
    m[0].xyzw * v.xyzw
  + m[1].xyzw * v.xyzw
  + m[2].xyzw * v.xyzw
  + m[3].xyzw * v.xyzw
  ```
  の転置が
  ```glsl
  vec4(dot(m[0],v), dot(m[1],v), dot(m[0],v), dot(m[0],v))
  ```
  で、右から掛ける
  ```glsl
    m[0].xyzw * v.xxxx
  + m[1].xyzw * v.yyyy
  + m[2].xyzw * v.zzzz
  + m[3].xyzw * v.wwww
  ```
  の転置は
  ```glsl
  vec4(dot(m[0],v.xxxx), dot(m[1],v.yyyy), dot(m[1],v.zzzz), dot(m[1],v.wwww))
  ```
  である。
  - 行列を転置するのは正規直行行列のアフィン変換でベクトルレジスタを節約するときに使う。
  </details>
