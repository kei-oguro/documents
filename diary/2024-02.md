02/29 木
--------

9:00の目覚ましで10:00起床。リモートなのに10:30スタートというのは慣れるべきではない。

水やりは13:00にペットボトル1本。

__コーヒー__ をプレスで淹れた。味がしない。

リモート出社。

__ランチ__ はサラダチキンとレタスを炒めたものを卵でとじたもの。

__晩飯__ は即席みそ汁に乾燥食材を入れたものとスナック菓子。

- __FX__ 日銀関係者の発言でドル円が結構下がった。昨晩小銭を取りに行ってれば結構なことになってた。結果的には1円を超える値動きで、参加してたら楽しめただろう。まあ、寝る前に仕込む気にはなりづらいし、月末の朝の下落はいつもの実需っぽくて、乗っかる気にはなれなかっただろうけど。

git で main ブランチをを今いるブランチのコミットに変更したいとき、`git checkout main` してから `git merge current` して、さらに戻りたければもう一度 `git checkout current`としていたが、直接 main の指すコミットを変更できるはずだとずっと思っていた。何故かそれが reset だと思っていてたどり着かなかったが、 branch サブコマンドを見たら `git branch -f main current` で良いと分かった。なるほど、とは思うが、こういう直交性の無さが嫌いだ。

リンゴをミルサーでジュースにして炭酸を入れて飲んだ。ミルサーは勢いが強過ぎてリンゴを液体にするには向かない。すりおろしリンゴは粘度が高くて、残った個体部分が落ちない。

--------

喉も傷めたし、最近部屋の湿度が低いのではないかと疑っていたのに調べようとしなかった。今日、ふと加湿器の湿度計を見たら60％になっていた。いつもは70%に張り付いているのに。70%表示は実際にはもっと高いだろう。それが70%よりは少なくなっている。音がうるさいのでいつもはおやすみモードにしているが、うるおいモードにしてみた。体感でハッキリと湿度が上がった。とりあえずおまかせにして様子をみることにする。

米国の利下げが遠のくニュースが多い。円高に転換したら FX でお小遣いが欲しいと思っていたが、落ち着いた値動きだと利益を取るのが難しいかも、何なら大損するかも知れない。それでも、しばらく順調に円高が進むようだと、日本の株価は落ちるのではないか。 __NISA__ の国内株ETFを日経平均ダブルインバースとかにポジションチェンジしたい。ギャンブルで。

食べてすぐ寝る荒業で、久しぶりに早く寝ることに成功。

  - 多過ぎた。冷凍餃子と豆腐は不要だった。
  - 豚肉は煮過ぎた。鶏団子が中まで出汁殻になっていないので、鶏団子、モヤシ、豚肉の順番が良かったかも。

__コーヒー__ をもう一杯マキネッタでカフェオレにした。パッキンを交換して初めて使う。ねじ回しの様子からすると、今回の方が厚みが薄い気がするが。

`三越本店`の地下で __コーヒーをテイクアウト__ 。売ってる豆を選んでドリップして貰える。

[Rustで危険なコードを書く方法](https://github.com/Speykious/cve-rs)だって。[ライセンスがおふざけ](https://github.com/Speykious/cve-rs/blob/main/LICENSE)で面白い。

- Dependency Injection のコンテナ作成の時の`AsSelf()`が何のことか、ようやく分かった。  
  ```cs
  builder.RegisterType<Type>.AsSelf().As<IType>();
  builder.RegisterType<Type>.As<Type, IType>();
  ```

寒かったのでまっすぐ帰宅。

__晩飯__ はザウアークラウトでシャウエッセンと冷凍餃子と卵を煮たもの。卵の白身がレシピ写真で見るポーチドエッグのように固まって、酸性で作る効果を知った。

- `Quake II RTX`で遊んでみた。 WASD 操作に慣れていないのでツラい。ウチの環境ではFPSが低いので3D酔いした。
  - https://www.nvidia.com/ja-jp/geforce/rtx-remix/ のAIアシストのによるテクスチャ改善は入っていなかった。勘違いしていた。
  - レイトレーシング凄いというシーンは見当たらなかった。探せば、影の形がなるほどレイトレだ、という場所は見つかる。武器を発射して、それがグローバルイルミネーションの光源になっているかも知れないが、見た感じ昔ながらのピクセルシェーダの点光源にしか見えない。

`Portal RTX`もやってみた。グローバルイルミネーション感が出てるが、何かこれ見よがしのギミックが欲しい。パズルは流石の楽しさ。
