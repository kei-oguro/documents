## 12/08 火

  * 一発で起きたがダラダラしていてギリギリの電車になった。
  * __水耕栽培__ の発芽状況があまりよろしくない。戻したジェルボールをどうしようか。
    * __パーライト__ は水の加減が分かりにくい。 __発芽__ と __中間移植__ が楽になるかと思ったが、発芽に向いていないかも知れない。
      * 嫌光性の問題で発芽していないかも知れないので、調子の悪い容器を __暗室__ へ。まあ、その容器には __溺れた__ っぽい __カモミール__ と __クレソン__ が横たわっているのだが。
    * __キッチンペーパー__ だけだと簡単に乾くので、暗室でないとケアできない。
      * これを、 __スポンジ__ の上にキッチンペーパーを置くことで改善を試みた。 __唐辛子__ シリーズと __エディブルフラワー__ シリーズはこの手法。今のところ最高の成績。
    * __ジェルボール__ に直接種まきというパターンも試してみたいが何で行こうか。 __ミント__ の寄せ植えなんてかわいらしくていいと思うけど、失敗続きだし、種が小さくて採りだすのが面倒だ。
    * 2度発芽させて2度本番移植までたどり着かなかった __小松菜__ をパーライトに直接種まきしてみよう。
  * ランチ前の __サラダ__ をコンビニで買って行く。
    * 駅の中のコンビニがローソンだったか、ファミマだったか。ファミマだと大根サラダが美味しいんだけど。
    * ローソンなら、駅の手前のセブンイレブンの方がいいが、ファミマであることに賭けて行ってみたらローソンだった。
    * キャベツは1つしかなかった。袋のキャベツ1つと、袋のレタスサラダ1つと、容器入りの海藻と大根サラダ1つ。
      * ローソンの海藻と大根サラダはそういえば中々だった。
  * __コーヒー__ はカメルーン100%に。
    * 今は結構粗めの設定なのか。 7→6 にしてみよう。
  * `sort|uniq` の結果がこんなことになるなんて、勘弁して欲しいんだけど
    * string 'm_Name' .. 'swirl'
    * string 'm_Name' .. 'Swirl'
    * string 'm_Name' .. 'swirl'
  * mingw ではなく windows の sort が呼ばれていたらしい。スラッシュのパスで動いたから油断してた。
  * __食べる順番ダイエット__ の __キャベツ__ のバルサミコ酢甘過ぎ問題は慣れてきたかも。
  * __ランチ__ は `鶏梵梵` で蒸し鶏よだれどり味。
    * 蒸し鶏は鶏肉が増えないのか。次回はカツじゃないチキンカレーを食べたい。黒にしようか赤にしようか。黒はホワジャオが効いていてインパクトがあるが、赤の方が完成度が高い。
  * 明日のキャベツを `ファミマ` で買った。大根サラダが美味しいのはいいんだけど、キャベツは130gで少し小さかった。
  * 水耕栽培用の __小瓶__ を手に入れるために、センター街の `日本酒原価酒場` に寄って2本 __テイクアウト__ したい。
  * 17:30 にして __お腹が減りそう__ な予感がある。困る。せめてプロテインバーの時間まで待ってもらえないだろうか。
  * 早めにプロテインバー食べた。ランニングと関係ない、ただの栄養だな。
    * 「トレーニング直後の30分がゴールデンタイムで、そのタイミングでタンパク質を摂取すると筋肉になりやすい」という説がある。しかし、そんなにすぐに消化するものか？そのタイミングで腸にあるとか、血中のプロテイン分解物(ペプチドの類)が多いのがいいという話なら、運動の1時間前とかがいいんじゃないの？胃からは送り出したくらい。
